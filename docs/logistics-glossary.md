## KPI Terms

| Term | Definition |
|------|-----------|
| **Plan Completion %** | Actual performance divided by planned target, expressed as percentage |
| **TEU Throughput** | Number of TEU units processed in a given period |
| **Dock Utilization** | Percentage of available dock time actively used for loading/unloading |
| **Deviation** | Difference between planned and actual values |
| **Rolling Average** | Average calculated over a moving time window (e.g., last 7 days) |
